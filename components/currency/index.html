shopkit.currency

Currency handling for django-shopkit. It comes in a simple and an advanced variant. The simple variant assumes a single currency throughout the webshop project, advanced currency support allows for using multiple currencies throughout the site.

Use a setting to define the way in which prices are formatted site-wide. This way we have a simple mechanism for formatting prices everywhere on the site while leaving everything loosely coupled.
