Settings¶

shopkit.currency.simple.settings.CURRENCY_MAX_DIGITS¶

Maximum number of decimals for PriceField. Defaults to: 6.

shopkit.currency.simple.settings.CURRENCY_DECIMALS¶

Number of decimals for PriceField. Defaults to: 2.

shopkit.currency.simple.settings.CURRENCY_FORMATTING¶

Formatting string for displaying monetary units.

For example, euro’s will be nicely rendered with:

SHOPKIT_CURRENCY_FORMATTING = u"\u20AC %.2f"
