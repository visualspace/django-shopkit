Settings¶

shopkit.category.settings

To set the values below from settings.py, prepend their names with SHOPKIT_. For example:

SHOPKIT_CATEGORY_MODEL = 'myapp.MyCategory'
shopkit.category.settings.CATEGORY_MODEL¶

Reference to the model class defining a category.

shopkit.category.settings.USE_MPTT¶

Whether or not to use django-mptt. This is enabled by default when mptt is in Django’s INSTALLED_APPS and disabled otherwise.
