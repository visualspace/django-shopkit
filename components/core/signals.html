Signals¶

shopkit.core.signals

shopkit.core.signals.order_state_change¶

Signal called whenever an order changes from one state to another.

Listeners should have the following signature:

def mylistener(sender, old_state, new_state, state_change, **kwargs):
    ...
Parameters:
  • sender – Order object
  • old_state – (raw value) of old state
  • new_state – (raw value) of new state
  • state_change – OrderStateChange pertaining to the state change
