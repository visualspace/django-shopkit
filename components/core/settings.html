Settings¶

shopkit.core.settings

To set the values below from settings.py, prepend their names with SHOPKIT_. For example:

SHOPKIT_PRODUCT_MODEL = 'myapp.MyProduct'
shopkit.core.settings.CUSTOMER_MODEL¶

Reference to the customer model used in the shop.

shopkit.core.settings.PRODUCT_MODEL¶

Reference to the model class defining the product.

shopkit.core.settings.CART_MODEL¶

The model used for shopping carts.

shopkit.core.settings.CARTITEM_MODEL¶

The model used for shopping cart items.

shopkit.core.settings.ORDER_MODEL¶

The model used for orders.

shopkit.core.settings.ORDER_STATES¶

Mapping for order states of the following form:

SHOPKIT_ORDER_STATES = (
    (00, _('Temp')),
    (10, _('New')),
    (20, _('Blocked')),
    (30, _('In Process')),
    (40, _('Billed')),
    (50, _('Shipped')),
    (60, _('Complete')),
    (70, _('Cancelled')),
)
shopkit.core.settings.DEFAULT_ORDER_STATE¶

Default state for new orders. By default, the first state in SHOPKIT_ORDER_STATES is selected.

shopkit.core.settings.ORDERSTATE_CHANGE_MODEL¶

Model used for logging state changes of orders.

shopkit.core.settings.ORDERITEM_MODEL¶

The model used for order items.

shopkit.core.settings.MAX_NAME_LENGTH¶

(Optional) The maximum name length for named products in the webshop. This defaults to 255.
